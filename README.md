# 🚀 ShopRocket Ops Analytics

**E-Commerce Operations & Supply Chain Analytics**  
Built on the [Olist Brazilian E-Commerce Dataset](https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce)

---

## 📌 Project Overview
Analyzing 100K+ orders to surface delivery bottlenecks, carrier SLA 
breaches, seller performance gaps, and demand trends across Brazilian 
e-commerce operations.

## 🛠️ Tech Stack
| Tool | Purpose |
|------|---------|
| Python (Pandas, Matplotlib, Seaborn) | EDA & visualizations |
| SQL (PostgreSQL) | Business queries & aggregations |
| Tableau Public | Interactive dashboard *(coming soon)* |

## 📁 Project Structure
| Folder | Contents |
|--------|----------|
| `notebooks/` | 4 analysis notebooks |
| `sql_queries/` | Business SQL queries |
| `reports/` | 7 exported charts |
| `dashboards/` | Tableau workbook *(WIP)* |
| `data/` | Dataset instructions (CSVs excluded) |
